# Flask Portfolio Website

A modern, responsive portfolio website built with Flask featuring a dynamic resume, project showcase, contact system, and admin dashboard.

### Core Features

- **Responsive Design**: Modern glass-morphism UI with smooth animations
- **Interactive Resume**: Dynamic resume with PDF download functionality
- **Project Showcase**: Filterable project gallery with detailed views
- **Contact System**: Rate-limited contact form with email notifications
- **Admin Dashboard**: Secure admin panel for content management
- **Blog System**: Markdown-based blog with article management
- **AI Chat Widget**: Interactive chatbot for visitor engagement

### Technical Highlights

- **Email System**: Contact form notifications with user confirmations
- **PDF Generation**: Dynamic resume PDF creation with ReportLab
- **Database Management**: SQLAlchemy ORM with Flask-Migrate
- **Authentication**: Flask-Login with JWT support
- **Rate Limiting**: Protection against spam with Flask-Limiter
- **WebSocket Chat**: Real-time chat using Flask-SocketIO
- **File Uploads**: Secure image upload with PIL optimization
